================================================================================
DeePAW 模型推理加速 - 工作报告
================================================================================

本目录包含 DeePAW 项目模型推理管线的性能优化工作记录。

优化工作分两轮进行，测试基准为 HfO2 结构（80x80x80 网格，512000 个探针点，
12 个原子），使用 24GB GPU (NVIDIA A10/A5000 级别)。

文件列表:
  1. round1_cpu_preprocessing.txt   - 第一轮优化：CPU 预处理加速（4.93x）
  2. round2_gpu_inference.txt       - 第二轮优化：GPU 推理端与传输优化
  3. external_suggestions_review.txt - 外部 16 条优化建议的逐条评估
  4. summary_and_future.txt         - 总结与未来优化方向

时间线:
  第一轮: CPU 预处理 19.2s -> 3.9s (4.93x 加速)
  第二轮: 传输/加载优化 + inference 模式 + batch size 可配置化

当前瓶颈: GPU 推理占总时间 ~80%（e3nn tensor product 为主）
